Comments

Commnets are a very useful tool used in web development. It helps to understand each section of the code by the developer. It makes the code look neat and clean. We can use ctrl+/ to write a commnet in HTML.In HTML, comments are used to add explanatory or descriptive text that is not displayed on the web page. Comments are useful for providing notes to yourself or other developers working on the code. They can also be helpful when you want to temporarily remove or disable a portion of your HTML code without deleting it entirely.
